(September - November 2020)
  • Performed tiled image convolution using CUDA C, CMake and Visual Studio.
  • Input image is blurred and darkened by convolution mask.

Extra Thoughts:

Most Fun Part of Project: Comparing the input image (left) to the resulting blurred image (right).
Biggest Challenges: Using the proper block size, calculating proper convoultion values in the kernel, and handling image boundaries.

  • Uses functional GUI, object-oriented concepts, software design patterns, and touch screen features.

Extra Thoughts:

Most Fun Part of Project: Playing the game! Also, I really enjoyed picking out the background music.

The Player Cyborg is displayed as a red square, with the Opponent Cyborgs being hollow red squares. The blue triangles represent Checkpoints, green circles represent Refuelling Stations, and purple hollow triangles represent Drones, which are obstacles.
